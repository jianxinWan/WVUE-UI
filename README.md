# 介绍
----

`WVUE-UI` 是一款基于 `Vue.js 2.0` 的前端 UI 组件库，主要用于快速开发网页

### 特性

- 基于 `Vue` 开发的 UI 组件
- 使用 npm + webpack + babel 的工作流，支持 ES2015
- 提供友好的 API，可灵活的使用组件

### 支持
- 手机端、电脑端适配
- 兼容ie8、chrome等主流浏览器。
